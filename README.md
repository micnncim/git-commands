# Git Commands

[![license][license-badge]][license]

A collection of Git custom subcommands.

## Commands

### `git prune-branch`

Deletes all the merged branches.

```console
$ git prune-branch
==> Deleted branch 'test1'
==> Deleted branch 'test2'
```
