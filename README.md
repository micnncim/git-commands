# Git Commands

A collection of Git custom subcommands.

## Installation

```
$ git clone https://github.com/micnncim/git-commands.git
$ cd git-commands
$ ./install.sh
```

## Commands

### `git delete-squash-merged-branches`

Deletes the squash-merged branches with inquiries.
